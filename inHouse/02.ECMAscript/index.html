JavaScript 매우매우 기초강의 정리페이지

    1. 변수

  • 1. 변수는 문자와 숫자, $와 _만 사용가능하다.
  • 2. 첫글자는 숫자가 될 수 없다.
  • 3. 예약어(alert, console.log등)은 사용할 수 없다.
  • 4. 가급적 상수(const)로 선언한 변수는 대문자로 선언한다.
  • 5. 변수명은 읽기 쉽고 이해할수 있게 선언한다.
  • let
    • 선언 후 차후에 바꿀 수 있는 변수
    • var의 단점을 수정하여 나온 변수이다.
  • const
    • 절대로 바뀌지 않는 상수
    • 수정할 수 없다.
    • 대문자로 선언하여 시인성을 좋게하는 편이다.
    • 처음에는 const로 선언하고 차후에 변경될 수 있는 변수는 let으로 변경해준다.

    2. 자료형

  • 문자형

    • 문자형은 ""(큰따옴표), ''(작은따옴표), ``(백틱)으로 쓸 수 있다. 3개의 큰차이는 없으나 통일성있게 하나만 쓰는게 좋다.
    • ``(백틱)은 문자형 내부에 변수를 넣을때 편리하다. --> console.log(`He is ${userName}`)꼴로 나타낼수 있다.

  • 숫자형

    • 숫자형은 더하기(+), 빼기(-), 곱하기(*), 나누기(/), 나머지(%)로 분류된다.
    • 숫자를 0으로 나누면 infinity(무한대)가 출력된다.
    • 문자열을 숫자로 나누면 NaN(Not a Number)가 출력된다.

  • boolean형

    • boolean형은 true(참), false(거짓) 두개 종류만 존재한다.

  • null/undefined

    • null : 존재하지 않는 값
    • undefined : 값이 할당되지 않음.

  • typeof 연산자

    • typeof 연산자는 해당 자료형이 어떤 종류인지를 알려준다.
    • console.log(typeof 7) // number
    • string은 문자를 뜻한다.
    • typeof null --> object(객체형)로 출력된다.
    • null은 객체가 아니다. JS의 오류이지만 아직 잡을 생각이 없다고 한다.
    • 문자형 + 문자형으로 해 새로운 문자형을 만들수 있다. 하나의 자료형이라도 문자형이면 사칙연산이 아닌 문자 붙여주는 기능을 수행하게 된다.
