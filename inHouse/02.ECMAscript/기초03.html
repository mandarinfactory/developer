JavaScript 기초강의 정리페이지03

    10.함수(function)

  • 익히 사용했던 console.log, alert(), confirm() 모두 함수이다.
  • function 함수명 (name//매개변수,parameter) {
    console.log(`Hello, ${name}`)
    } 꼴로 함수는 쓸수 있다. 함수명(Brent)
  • 매개변수는 없을수도 있고 한개일수도, 여러개일수도 있고 여러개라면 ,로 구별한다.
  • 함수는 유지보수가 쉽다.
  • 함수내/외에서 각각 선언한 변수는 설령 그 변수명이 같더라도 충돌하지 않고 할당된 값을 각각 출력시킨다.
  • 함수외에서 선언된 변수는 전역변수, 함수내에서 선언된 변수는 지역변수라고 한다.
  • 따라서, 전체서비스에서 공통으로 쓰이는건 전역변수로 쓰되, 왠만하면 지역변수로 각각 함수에다가 선언하는게 차후에 충돌을 방지할수 있다.

    11. 함수표현식, 화살표함수(arrow function)

  • 함수선언문 : 어디서든 호출이 가능하다.
    다시 말해 함수가 선언되기 전에 함수를 호출해도 상관없이 실행된다. (함수호이스팅 문제!)
    sayHello();
    function sayHello () {
    console.log('Hello');
    }
  • 함수표현식 : 코드에 도달하면 생성된다. 함수선언문처럼 어디에든 호출하면 안된다.
    let sayHello = function () {
    console.log('Hello');
    }
  • 화살표함수 : 함수를 보다 간결하게 작성하게 해준다.
    let sayHello = () => {
    console.log('Hello')
    }
